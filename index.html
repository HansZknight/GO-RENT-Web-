Pertanyaan yang sering ditanyakan tentang GO-RENT.

GO-RENT adalah aplikasi rental kendaraan berbasis Java Console yang dirancang untuk pembelajaran pemrograman dasar. Aplikasi ini mendemonstrasikan konsep OOP, array, conditional, looping, dan input handling di Java.
  1. Pastikan Java JDK 8+ sudah terinstall
  2. Clone repository dari GitHub
  3. Compile dengan: javac TBPDDPUAS.java
  4. Jalankan dengan: java TBPDDPUAS
